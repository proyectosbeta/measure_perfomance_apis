# Deno_oak

API with Deno and OAK.
Data cars: https://www.back4app.com

## Tecnologies

* Deno 1.31.1
* Typescript 4.9.4
* Oak

## Install

```bash
deno run index.ts
```

### Run

```bash
deno run --allow-net --allow-env --allow-read server.ts
```

## Use API

### GET ALL

```bash
localhost:3000/api/v1/cars
```

### GET

```bash
localhost:3000/api/v1/cars/cptB1C1NSL
```
